`POST /greeting`

Creates an greeting

> Body parameter

> Successfull response

```json
{
  "greeting": "Request!"
}
```

<h3 id="get-greeting-parameters">Parameters</h3>

|Parameter|In|Type|Required|Description|
|---|---|---|---|---|
|body|body|object|true|Contains the greeting to create|
|» greeting|body|string|true|none|

> Example responses

> Successfull response

```json
{
  "greeting": "Response!"
}
```
